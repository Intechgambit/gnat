# Genomic Analytics Tool (gnat)
## Tool describtion: 
GNAT is a python Fet supported desktop App developed by Intechgambit to process genomic data (DNA and Edna) by pulling data from [NCBI Entrez databased](https://www.ncbi.nlm.nih.gov/search/) utlizing the Entrez API, creating a custom blast supported database, making taxonmic assignments and annonotations and generating analytics and vizulizations.
